<html><p style="text-indent: 0em;"><b>POST COMMANDS</b>: 
</p>
<hr>
<p style="text-indent: 0em;"><b>PostUsers</b>: 
POST /users {parameter list: loginName, loginPassord, username, password, email, fullname(optional)}  
	  Add a user to the User Repository. </p>
<hr>
<p style="text-indent: 0em;"><b>PostProjects</b>: 
  	POST /projects  {parameter list: loginName, loginPassord, Latitiude, longitude, name, price} 
	  Add a Project to the Project repository.</p>
<hr>
<p style="text-indent: 0em;"><b>PostConsultants</b>: 
  	POST /consultants {parameter list: loginName, loginPassord, name, priceHour, bonus (optional)}
	  Add a consultant to the Worker Repository</p>
<hr>
<p style="text-indent: 0em;"><b>PostWorkerInProject</b>: 
  	POST /projects/{pid}/{type} {parameter list: loginName, loginPassord, WorkerId}
	  Add a consultant or Manager to a project/subproject</p>
<hr>
<p style="text-indent: 0em;"><b>PostSubprojects</b>: 
  	POST /projects/{pid}/subproject {parameter list: loginName, loginPassord, ProjectId}
	  Add a subproject to a project/subproject.</p>
<hr>
<p style="text-indent: 0em;"><b>GET COMMANDS</b>: 

  GET COMMANDS: {parameter list: accept, output-file(optional)}</p>
<hr>
<p style="text-indent: 0em;"><b>GetUsers</b>: 
  	GET /users : Return the information of all users in the User Repository</p>
<hr>
<p style="text-indent: 0em;"><b>GetUser</b>: 
  	GET /users/{username}  : Return the information of the user with the  specify {@code username} of the User Repository</p>
<hr>
<p style="text-indent: 0em;"><b>GetProjects</b>: 
  	GET /projects/{pid}: Return the information of the project with the  specify {@code ProjectId}</p>
<hr>
<p style="text-indent: 0em;"><b>GetProjectWorkers</b>: 
  	GET /projects/{pid}/{type} : Return the information of all consultants or of the Manager of a project with the  specify {@code ProjectId}</p>
<hr>
<p style="text-indent: 0em;"><b>GetSubproject</b>: 
  	GET /projects/{pid}/subproject : Return the information of all subprojects of a project with the  specify {@code ProjectId}</p>
<hr>
<p style="text-indent: 0em;"><b>PATCH COMMANDS</b>: 
</p>
<hr>
<p style="text-indent: 0em;"><b>PatchUser</b>: 
  	PATCH /users/{username} {parameter list: loginName, loginPassord, oldpassword, newPassword} 
	  Updates the password of the user identified by the  specify {@code username}. </p>
<hr>
<p style="text-indent: 0em;"><b>PatchProject</b>: 
  	PATCH /projects/{pid} {parameter list: loginName, loginPassord, Latitiude, longitude, name, price (the last four optional)} 
	  Update the information of the project identified by the  specify {@code ProjectId}.</p>
<hr>
<p style="text-indent: 0em;"><b>PatchConsultant</b>: 
  	PATCH /consultants/{cid} {parameter list: loginName, loginPassord, name, priceHour (the last two optional)} 
	  Updates the information of the consultant with the  specify {@code WorkerId}. </p>
<hr>
<p style="text-indent: 0em;"><b>DELETE COMMANDS</b>: 
</p>
<hr>
<p style="text-indent: 0em;"><b>DeleteProjects</b>: 
  	DELETE /projects/{pid} {parameter list: loginName, loginPassord} 
	  Deletes the project with the  specify {@code ProjectId} and all its subprojects</p>
